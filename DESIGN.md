# DESIGN.md — 高阶个人简历网站

## 产品定位

**专业求职者个人简历网站**，采用现代玻璃拟态设计风格，营造高级、沉浸式的视觉体验。目标用户：HR、技术面试官、潜在合作者。

**设计目标**：
- 第一印象：高级、现代、专业
- 视觉冲击：渐变光效、玻璃拟态、精致动效
- 体验性：流畅的交互、暗色模式沉浸感
- 工具性：支持一键下载 PDF（打印样式优化）

## 气质与意象

**关键词**：科技感渐变、玻璃拟态、发光效果、现代极简、沉浸式体验。

**具象场景**：
- 深邃的夜空背景，漂浮的光晕粒子
- 头像周围脉动的光环，呼吸般的光晕
- 玻璃质感的卡片，悬停时微微上浮并发光
- 时间线上的渐变连接线，指向职业里程碑

## 配色方案

| 用途 | 颜色 | 意象 |
|------|------|------|
| 主渐变 | `#667eea` → `#764ba2` | 科技蓝紫渐变，高级感 |
| 强调-青 | `#00f5d4` | 薄荷绿，清新点缀 |
| 强调-粉 | `#f093fb` | 柔粉，温暖细节 |
| 强调-金 | `#f5af19` | 金色，重要标注 |
| 背景-暗 | `#0f0f23` | 深夜背景，沉浸感 |
| 背景-亮 | `#f0f2f5` | 亮色模式背景 |
| 背景-玻璃 | `rgba(255,255,255,0.08)` | 玻璃拟态卡片 |
| 文字-主 | `#ffffff` | 暗色模式主文字 |
| 文字-次 | `rgba(255,255,255,0.85)` | 次级文字 |
| 文字-弱 | `rgba(255,255,255,0.6)` | 辅助文字 |

**配色口诀**："**蓝紫渐变是魂，薄荷青是睛，深夜是背景，玻璃是衣**"。

## 字体排版

- **标题**：`Inter`（现代无衬线，清晰有力）
- **中文正文**：`Noto Sans SC`（简洁易读）
- **数字**：`Inter` 配合 `font-weight: 800`（统计数字突出）
- **字号层级**：
  - Hero 姓名：`3.5rem`（醒目）
  - 区块标题：`2.5rem`
  - 卡片标题：`1.2rem`
  - 正文：`1rem`

## 动效与交互

- **背景光晕**：缓慢漂浮动画（8-10s），营造氛围
- **头像光环**：脉动呼吸动画（3s），吸引注意力
- **入场动画**：从下方淡入上移（0.7s）
- **卡片悬停**：上浮 + 发光 + 渐变边框
- **按钮光效**：点击时扫光动画
- **进度条**：渐变填充 + shimmer 微光
- **暗色切换**：图标 360° 旋转

## 视觉特效

| 特效 | 实现 | 应用位置 |
|------|------|----------|
| 玻璃拟态 | `backdrop-filter: blur(20px)` + 半透明背景 | 导航栏、卡片、统计框 |
| 渐变发光 | `box-shadow` + 渐变色 | 按钮、卡片悬停、头像 |
| 背景光晕 | `radial-gradient` + `animation` | Hero 区、页面背景 |
| 文字渐变 | `background-clip: text` | 标题、数字、图标 |

## 页面结构

| 区块 | 设计要点 |
|------|---------|
| 1. 导航栏 | 玻璃拟态 + 滚动变色 + 渐变高亮 |
| 2. Hero | 动态光晕背景 + 呼吸光环头像 + 居中布局 |
| 3. 统计卡片 | 玻璃质感 + 渐变数字 + 悬停发光 |
| 4. 时间线 | 渐变连接线 + 发光徽章 + 玻璃卡片 |
| 5. 技能 | 进度条渐变 + shimmer 微光 |
| 6. 作品 | 大卡片 + 悬停上浮 + 渐变遮罩 |
| 7. 联系 | 玻璃表单 + 发光输入框 |

## 响应式策略

- **电脑（≥992px）**：完整效果，时间线左右交替
- **平板（768-991px）**：保持玻璃效果，时间线单列
- **手机（<768px）**：简化光晕，核心动效保留

## 设计禁忌

- ❌ **不**用纯色背景（必须有渐变或光晕）
- ❌ **不**用厚重边框（保持轻盈玻璃感）
- ❌ **不**用过于鲜艳的色块（保持克制高级）
- ❌ **不**做过多闪烁动画（保持流畅优雅）
- ❌ **不**用扁平设计（必须有层次和光效）
